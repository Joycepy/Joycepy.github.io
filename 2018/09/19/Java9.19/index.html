Java9.19

java virtual machine ：Java所定义的虚拟CPU
提供以下定义：
指令集、寄存器组、类文件格式、栈、堆（实现内存动态分配、自动回收）

JRE:Java运行环境（jvm+核心类）
Bytecode 字节码
JDK java developer’s kit=JSDK software development kit
java开发平台：Javase\javaee\javame

java程序类型：
application（可独立执行）
applet(小程序，嵌入网页 执行，需要java兼容浏览器)
servlets（服务器执行，桥梁）

applet小程序：
init()初始化，start()开始执行，stop()停止执行
destroy()释放资源，paint()绘制区域

Servlet:
超文本文件
Tomcat

JSP全名为Java Server Pages
