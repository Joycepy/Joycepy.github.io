执行SQL语句

主要分三步：创建command，绑定参数（可选），调用SQL执行方法

所有从数据库取得的数据都被表现为字符串。使用绑定参数的方法可以防止 SQL 注入攻击。在 SQL 语句中，可以嵌入一个或多个参数占位符。 一个参数占位符是以冒号开头的字符串。

bindValue()：绑定一个参数值
bindValues()：在一次调用中绑定多个参数值

可以使用不同参数多次执行。

执行非查询语句，即不取回数据的语句，调用excute()方法，返回执行 SQL 所影响到的行数。
