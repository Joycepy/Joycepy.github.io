remove-nth-node-from-end-of-list

发表于 2019-11-01

题目描述

对于给定的链表，删除倒数第n个节点，返回头节点。

解题思路

一个指针遍历两次，或者两个指针遍历一次。对于两个指针p,q，需要固定间隔长度n (p-q=n)，然后同时移动两个指针，直至q移到链表尾。此时q指向需要被移除的节点。注意返回空指针的处理。

易错情况：

([1,2],1) ([1,2],2) ([1],1)

编译时报错：error: expected unqualified-id before 'while'

原因是在类的定义中误写了循环体。
