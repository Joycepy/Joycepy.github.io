关键字struct

c++ 的struct 声明变量的时候不需要加struct前缀。在C语言中，实例化一个结构体必须要加上关键字struct。为了避免每次用的时候都要加上struct前缀，可以定义的时候加上typedef。

enum枚举类型

第一个枚举成员的默认值为整型的 0，后续枚举成员的值在前一个成员上加 1,可以在定义枚举类型时改变枚举元素的值。在C 语言中，枚举类型是被当做 int 或者 unsigned int 类型来处理的。
